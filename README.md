
# CRUD_Coursera

Simple CRUD application.


## Run Locally

Clone the project

```bash
  git clone https://github.com/https://github.com/JLuis31/CRUD_Coursera.git
```

Go to the project directory

```bash
  cd CRUD_Coursera
```



## Tech Stack


**Server:** C#, .NET


## Features

- Api Post
- Api Get
- Api Patch
- Api Delete
